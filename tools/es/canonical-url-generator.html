¿Qué es una URL Canónica?

        
Una URL canónica indica a los motores de búsqueda cuál es la versión "maestra" o preferida de una página cuando hay múltiples URLs con contenido similar o duplicado. Esto ayuda a prevenir problemas de SEO relacionados con contenido duplicado.
