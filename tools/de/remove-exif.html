Über EXIF Daten

EXIF (Exchangeable Image File Format) Daten sind Metadaten, die in Bilddateien gespeichert sind. Sie können enthalten:

  • GPS Koordinaten (genauer Standort, wo das Foto aufgenommen wurde)
  • Kamera-/Handymodell und Hersteller
  • Datum und Uhrzeit der Aufnahme
  • Kameraeinstellungen (Blende, Verschlusszeit, ISO)
  • Vorschaubilder

Die gesamte Verarbeitung erfolgt lokal in Ihrem Browser - Ihre Bilder werden nie auf einen Server hochgeladen.
