# Multi-Template System for 205-Tools

## Overview
نظام قوالب متعدد لتحويل كل أداة لتجربة مستخدم فريدة.

## Templates Created

### 1. Image Converter Template
**File:** `templates/image-converter.html`
**لمن:** محولات الصور (JPG→PNG, PNG→WebP, etc.)
**المميزات:**
- معاينة مباشرة للصورة
- تحكم بالجودة (Slider)
- اختيار الصيغة (Format buttons)
- Before/After comparison
- 3 خطوات: Upload → Adjust → Download

### 2. PDF Tool Template  
**File:** `templates/pdf-tool.html`
**لمن:** أدوات PDF (Merge, Split, Compress, etc.)
**المميزات:**
- نمط iLovePDF
- رفع متعدد الملفات
- ترتيب بالسحب (Drag & Drop)
- خطوات واضحة (1-2-3)
- Google Drive / Dropbox integration (UI)

### 3. Text Analyzer Template
**File:** `templates/text-analyzer.html`
**لمن:** محللات النصوص (Word Counter, etc.)
**المميزات:**
- تحليل فوري Real-time
- إحصائيات تفاعلية ( animate numbers)
- لوحة جانبية للإحصائيات
- أزرار سريعة (Paste, Clear, Copy)
- Reading/Speaking time

### 4. Code Editor Template
**File:** `templates/code-editor.html`
**لمن:** أدوات المطورين (JSON Formatter, HTML Encoder, etc.)
**المميزات:**
- Syntax highlighting (highlight.js)
- Format/Minify/Validate buttons
- Status badges
- Dark/light theme support
- Tabs for multiple modes

### 5. Calculator Template
**File:** `templates/calculator.html`
**لمن:** الحاسبات (BMI, Loan, Mortgage, etc.)
**المميزات:**
- حقول إدخال منظمة
- نتيجة واضحة مع visualization
- Radio buttons & sliders
- Auto-calculate option
- Info/explanation section

### 6. Generator Template
**File:** `templates/generator.html`
**لمن:** المنشئات (Password, UUID, QR, etc.)
**المميزات:**
- Result display مع strength bar
- Length slider
- Checkbox options
- History tracking
- Copy & Regenerate buttons
